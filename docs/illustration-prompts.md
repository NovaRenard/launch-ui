# Промпты для генерации иллюстраций Flow AI

Стиль проекта: тёмный фон, фиолетово-синие акценты, минимализм, AI/tech эстетика.
Рекомендуемые инструменты: Midjourney, DALL·E 3, Ideogram, Adobe Firefly.

---

## 1. Hero — главная страница

**Место:** За или рядом с заголовком в секции Hero  
**Концепция:** AI-агент в процессе работы — диалоги, данные, CRM

### Midjourney / DALL·E 3:
```
Isometric 3D illustration of an AI agent processing sales conversations.
Multiple chat bubbles flowing into a central glowing neural node,
then routing to a CRM dashboard. Dark background (#0a0a0f),
purple and blue gradient accents (#7c3aed, #3b82f6).
Clean, minimal tech aesthetic. No text. 16:9 ratio.
--style raw --v 6
```

### Ideogram:
```
Dark tech illustration: abstract AI sales agent. Chat messages from WhatsApp
and Telegram icons converging into a central AI brain (glowing purple orb),
with arrows flowing to a CRM grid dashboard on the right.
Dark navy background, violet-blue color palette, isometric flat style, no text.
```

```
Dark background, purple-to-blue gradient on the phone screen.
No shadows, flat isometric style. Square format 1:1.
--style raw --v 6
```

### Шаг 2 — AI обрабатывает:
```
Minimal 3D icon: abstract neural network / brain with pulsing connections.
Floating geometric nodes connected by glowing lines.
Dark background, violet accent (#7c3aed). Square 1:1. No text.
--style raw --v 6
```

### Шаг 3 — Передача менеджеру:
```
Minimal 3D icon: document/lead card with a green checkmark,
floating above a hand receiving it. Dark background, emerald green accent.
Clean flat style. Square 1:1. No text.
--style raw --v 6
```

## 4. О нас — аватары команды

**Место:** В карточках команды вместо заглушек с инициалами  
**Концепция:** Стилизованные иллюстративные портреты в единой манере

### Серия аватаров (генерировать по одному):
```
Stylized illustrated portrait avatar of a [male/female] tech founder.
Abstract geometric face, flat vector style, no realistic features.
Background: circular gradient from [violet/blue/emerald/rose] to darker shade.
Minimal, modern, professional. Used as a team member avatar. Square 1:1.
No text. Style: Notion-style illustrated avatar.
```

**Варианты цветов по персонажам:**
- Алексей: `violet to purple` (#7c3aed → #6d28d9)
- Дарья: `blue to indigo` (#3b82f6 → #4338ca)
- Иван: `emerald to teal` (#10b981 → #0d9488)
- Мария: `rose to pink` (#f43f5e → #ec4899)

## Общие параметры стиля

При генерации всегда добавляй в промпт:
- `dark background`
- `no text, no labels`
- `minimal flat / isometric style`
- цвета: `#7c3aed` (violet), `#3b82f6` (blue), `#10b981` (emerald), `#f59e0b` (amber)
- формат зависит от места: `1:1` для иконок, `4:3` для карточек, `16:9` для hero

## Рекомендуемые ресурсы

- **Midjourney** — лучший для изометрии и абстрактных иллюстраций
- **Ideogram** — хорошо понимает стиль «flat tech dark»
- **Storyset / unDraw** — готовые SVG-иллюстрации (бесплатно, есть dark mode)
- **Humaaans** — для аватаров команды в абстрактном стиле
